# AgentVault AI

### Autonomous AI Investment Agent on Mantle

AgentVault AI is an autonomous portfolio intelligence agent built on Mantle Network that combines smart-money tracking, AI reasoning, market intelligence, and on-chain execution.

The platform continuously monitors whale activity, market sentiment, and ecosystem signals to generate conviction scores, recommend portfolio actions, and execute transparent rebalancing strategies on Mantle.
